Контекст:
  Rails приложение находится в директории: test_ops/

Цель проекта:
  Реализовать авторизацию пользователей sign in без регистрации.

Текущее состояние:
- Rails проект уже создан
- Devise установлен и настроен
- создана модель User
- пользователь создан вручную через rails console:
  email: test@example.com
  password: password123
- Devise views сгенерированы
- регистрация отключена

Задача:
Проверить и при необходимости исправить реализацию авторизации.

Требования:
1. Должен работать вход в систему:
   - пользователь вводит email и password
   - после успешного входа происходит переход на /home

2. Корневая страница должна быть защищенной:
   - /home доступен только авторизованным пользователям
   - неавторизованный пользователь должен редиректиться на страницу логина

Constraints:
- не добавлять регистрацию
- использовать только Devise

Expected output:
- рабочая авторизация 
- корректные routes
- защищённый контроллер Home

Проверка:
- пользователь не может зайти на /home без логина
- после логина доступ открыт
